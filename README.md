# Doc Teex

This doc need to be build and open in browser.
But cant be read as markdown file in `doc/mds/`

## Build it

### install mkdocs
unix based OS 
```
pip3 install Markdown
```


### build doc
```
./build.sh
```

### Open doc
```
./open.sh
```
_if ./open.sh dont work; just open ./doc/site/index.html in your browser_
